# ScFiler

Scalaで作っているファイラ

| キー | 説明 |
| --- | --- |
| ? | ヘルプ |
| gg | カーソルを先頭行に移動 |
| G | カーソルを最終行に移動 |
| j | カーソルを下に移動 |
| k | カーソルを上に移動 |
| h | 親ディレクトリに移動 |
| l | 子ディレクトリに移動 |
| SPC | マークを付ける/外す |
| v | 全てのマークをトグル |
| V | ヴィジュアルモード |
